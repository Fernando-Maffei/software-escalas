App de Escalas

Como esta instalacao vai funcionar?

Escolha como este computador deve abrir o sistema. Se quiser o modelo com uma maquina principal e outras se conectando pela rede, use os modos Servidor da rede e Cliente da rede.

Dados desta instalacao

Escolha uma pasta para guardar as configuracoes locais do sistema. No modo servidor, esta pasta pode ficar no disco C desta maquina. Se voce quiser padronizar backups e configuracoes, tambem pode compartilhar essa pasta no Windows depois.

Use um caminho local. O aplicativo cria a pasta automaticamente se ela ainda nao existir.
Use a mesma porta em todas as maquinas clientes. O padrao recomendado e 3000.

Conexao com a maquina servidora

Informe o endereco da maquina que vai hospedar o sistema. Exemplo: http://192.168.0.10:3000. Se a outra maquina estiver na mesma rede, os dois computadores usam o mesmo servidor.

Voce pode informar IP, nome da maquina ou URL completa. O aplicativo normaliza isso automaticamente.
