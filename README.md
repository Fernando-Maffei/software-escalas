# Download do Setup

Este branch existe apenas para distribuir o instalador Windows do projeto.

## Arquivo disponivel

- `AppDeEscalasSetup-1.1.0-beta.exe`

## Como usar

1. Baixe o arquivo `.exe`
2. Execute o instalador no Windows
3. Abra o aplicativo e escolha o modo desejado:
   `Nesta maquina`, `Servidor da rede` ou `Cliente da rede`

## Observacao

Quem precisa do codigo-fonte completo deve usar a branch principal do projeto.
